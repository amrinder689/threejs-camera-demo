Notes
- No build step required; this is a static demo.

Camera Controls
- Orbit-style controls (recommended):
	- Use `OrbitControls` to allow rotating, zooming and panning around the scene.
	- Left-click drag: rotate/orbit the camera around the target.
	- Scroll (or middle-click): zoom in/out.
	- Right-click drag (or Ctrl+drag): pan.

- Mobile / touch hints:
	- Pinch to zoom and swipe to rotate (when touch controls are enabled).
